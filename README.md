# sample-mvvm-retrofit


#### 📌 주요기능

- MVVM 아키텍처 패턴 사용  
- retrofit 사용한 네트워크 통신  
- recyclerView 활용  
- kotlin 언어로 작성  

#### 📌 간단소개  

- 한국영화데이터베이스(KMDb)에서 공공기관 영화 상세 정보 조회 API를 활용한 영화 검색 앱이다.  
- 입력란에 키워드를 입력하면 관련된 영화 정보가 RecyclerView에 그려지게 된다.  
- MVVM 패턴 + Retrofit + RecyclerView 조합을 Kotlin으로 개발했다.
